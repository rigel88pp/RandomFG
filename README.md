# RandomFG

こちらのアプリケーションは、啓林館出版の以下の書籍を所持していることを前提とした、**身内ノリ用ツール**です。

- 『Focus Gold 数学Ⅰ＋A 5th Edition』
- 『Focus Gold 数学Ⅱ＋B＋ベクトル（数学C）5th Edition』

※『Focus Gold 数学Ⅰ』などの分冊版でも問題番号は対応していると思われますが、**ページ番号は上記二冊に準拠**している点にご注意ください。

---

## 概要

設定画面で選択した単元の中から、難易度 ★2〜★3 の「求値問題」のみを対象としてランダムに出題します。

---

## 機能

- 単元選択による出題範囲のカスタマイズ
- 難易度 ★2〜★3 の問題のみを抽出
- ランダム出題
- **競技モード**
  - 単元を固定化
  - 出題開始からの経過時間を測定

---

## 注意事項

- 本アプリは **Focus Gold を所持していることを前提**としています。
- 問題文や解答は本アプリには含まれていません。
- あくまで **番号管理・ランダム抽出用の補助ツール**です。
